The Geeks

Esta presentación es el proyecto de un viaje de mil millas que recién comienza.

✨ El objetivo de este proyecto

Practicar las nuevas herramientas y funciones que hemos aprendido a lo largo de esta semana de este Bootcamp. con el fin de tener el mayor de número de preguntas y dudas y resolverlas practicando.

✨ Requisitos

  1. Crear una página en HTML y CSS
  2. Ejecutar comandos de Git para poder compartir código entre colaboradores usando GitHub.
  3. Cuidar la estructura de la página, códigos de calidad, estilos y activar GitHub Pages.
  4. Trabajar en equipo y practicar la comunicación activa, y hablar en público.
